How did viewing a diff between two versions of file help you see the bug that was introduced?

diff allows us to compare two files i.e the lines that have been changed or added.it shows lines existed in the old version with '-' sign and lines existed in the new version with '+' sign, while rest of the code remains the same.in this way we can find the bug easily comparing the lines that have been changed or edited and we don't need to check the full code of both the versions of file.
